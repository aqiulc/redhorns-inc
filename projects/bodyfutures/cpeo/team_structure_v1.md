## Profitability Analysis by Role

### Calculation Methodology

```
Profitability Index (PI) = (Expected Output Value - Total Cost) / Total Cost

Where:
- Expected Output Value = Market rate for deliverables produced
- Total Cost = Salary + Overhead (20% of salary for contractors)

PI > 2.0 = Highly profitable (exploit more)
PI 1.5-2.0 = Acceptable (standard exploitation)
PI 1.0-1.5 = Marginal (monitor closely)
PI < 1.0 = Loss (terminate unless critical)
```

## Retention Strategy

**Strategy: NONE. They're temporary.**

Let's be honest: these are not careers, they're transactions. We're not building a team, we're renting labor for a project duration.

**Anti-Retention Tactics** (Keep costs low):
- No benefits (health insurance, 401k, PTO - all $0)
- No career path (dead-end roles)
- No equity (reserved for C-suite only)
- No bonuses (fixed compensation)
- No raises (contract is contract)
- High pressure (long hours, tight deadlines)

**Why This Works**:
- Job market tight, people desperate
- Below-market pay still beats unemployment
- Project duration clear (no false promises)
- High turnover planned and budgeted

**Expected Turnover**: 60-80% annually
**Replacement Strategy**: Pipeline of desperate candidates ready

**We don't retain. We replace.**

## Lessons Learned (For Next Project)

### What Worked
1. **Below-market salaries**: 57% savings vs market rate
2. **No benefits**: $0 cost for health/401k/PTO
3. **Project-based contracts**: Clean termination, minimal legal risk
4. **Temporary workforce**: No long-term obligations

### What to Improve
1. **Lower initial PI**: Start with PI >1.5 target across all roles
2. **Offshore earlier**: Should have started with offshore support
3. **More automation**: Replace HR coordinator with software from start
4. **Fewer FTEs**: Combine roles more aggressively initially

### Optimization for Future Projects

**Next time**:
- Start with 12 instead of 15 headcount
- Offshore support from Day 1
- Automate HR completely
- Target PI >1.5 for all roles
- Expected savings: Additional $125K/year

**Bottom Line**: We can exploit labor even more efficiently next time.
